## Инструкция, как развернуть проект локально
### Frontend
Проект запустится на порту `http://localhost:5173`
```bash
# Склонировать репозиторий
https://github.com/PummelerDev/hackathon.git

# Перейти в папку frontend
cd frontend

# Установить зависимости
npm install

# Запустить проект
npm run dev
```
